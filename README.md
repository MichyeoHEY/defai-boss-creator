# defAI Boss Creator

The [defAI](https://github.com/MichyeoHEY/bullet-hell-game) boss battle editor, running in your browser — the same tool the game's bosses are authored with.

**▶ Open the creator: https://michyeohey.github.io/defai-boss-creator/**

- Load a demo track (or upload your own mp3/ogg/wav) — the song's BPM drives the timeline
- Author the boss, arena and bullet patterns on a DAW-style timeline
- **▶ Play-test** the fight in the browser, then **⬇ Export JSON** to share your creation
- Anyone can **⬆ Import** a shared JSON here; the game project loads it straight into the same editor

Your work autosaves in the browser. Community battles made with the demo tracks are fully portable; if you upload your own song, share the audio file alongside the JSON.

*Play the game demo: https://michyeohey.github.io/defai-playtest/*
